CPU


L'unità di elaborazione centrale (central processing unit, sigla CPU) è un tipo di microprocessore digitale general purpose che si contraddistingue per sovrintendere a gran parte delle funzionalità del computer digitale È detta unità "centrale" perché coordina in maniera centralizzata tutte le altre unità di elaborazione presenti nelle architetture hardware dei computer per la gestione delle varie periferiche interne o schede elettroniche (scheda audio, scheda video, scheda di rete)



RAM

La memoria ad accesso casuale o RAM (Random Access Memory) è un tipo di memoria del computer che consente l'accesso illimitato a qualsiasi parte dei dati memorizzati, e ha velocità relativamente elevate.



Refresh rate

La Refresh Rate e' un valore che indica quante volte in un secondo lo schermo aggiorna l’immagine, si misura in Hertz.
